# 국자와 연필


<img width="200" height="200" alt="Image" src="https://github.com/gminpark0117/new_cooking_diary/blob/master/assets/splash/splash.png" />
나만의 레시피를 관리하고, 요리 과정을 기록하며, 장보기에도 도움을 줄 수 있는 다이어리 앱입니다.

---

##  주요 기능

### 1. 레시피 관리 
-  레시피의 이름과 분량 및 소요시간, 필요 재료, 조리 순서를 입력할 수 있어요. 추가적인 메모와, 전체 레시피 및 각 순서에 대한 사진도 첨부가 가능합니다. 등록한 레시피들은 한눈에 리스트로 보거나 각 레시피별로 자세히 볼 수도 있어요.
- 외부 레시피 DB와 연결하여 유사도 검색을 통한 추천 레시피를 제공해요.

### 2. 요리 일기 
-  요리한 결과물에 대한 생각을 사진과 함께 로그로 남길 수 있어요. 등록한 로그들은 한눈에 리스트로 보거나 각 로그별로 자세히 읽을 수도 있어요. 

### 3. 장바구니 (Grocery Cart)
- 요리에 필요한 재료를 추가/삭제할 수 있는 장바구니 기능을 제공해요. 이미 구입한 재료들은 줄을 그어 표기할 수 있어요.
- 레시피별 자세히 보기에서 장바구니에 바로 재료를 추가할 수 있어 관리가 간편합니다.
